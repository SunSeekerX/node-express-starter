# node-express-starter

## 📌 简介

NodeJS Express api 项目启动模板。集成常用依赖，以及提交代码格式化、代码校验等。

- `chalk` 彩色控制台打印
- `cookie-parser` cookie 解析
- `cors` 允许跨域访问
- `internal-ip` 获取内网 ip
- `morgan` 请求日志中间件



## 📌 快速上手

**克隆项目**

```shell
git clone https://github.com/SunSeekerX/node-express-starter.git
```

**进入项目目录**

```shell
cd node-express-starter
```

**安装依赖**

```shell
yarn
# or npm
npm i
```

**启动运行**

```shell
yarn serve
# or npm
npm run serve
```
